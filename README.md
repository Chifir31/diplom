## Класс ComparisonOfFormulations

**Тест Б1 (положительный)**
- Описание: Проверка корректности работы метода, находящего схожие и идентичные формулировки.
- Метод: *find_similar_formulations_v1(first_formulations, second_formulations)*.
- Входные данные: *first_formulations*, *second_formulations*
```
first_formulations = ["Языки, утилиты и среды программирования, средства пакетного выполнения процедур",
                      "Интерфейсы взаимодействия внутренних модулей программного проекта"]
  
second_formulations = ["Языки, утилиты и среды программирования, средства пакетного выполнения процедур",
                       "Интерфейсы взаимодействия внутренних модулей программного продукта"]
```
- Ожидаемый результат: *true_s* - схожие формулировки, *true_i* - идентичные формулировки.
```
true_s = [{"form1": "Интерфейсы взаимодействия внутренних модулей программного проекта",
          "form2": "Интерфейсы взаимодействия внутренних модулей программного продукта"}]
true_i = ["Языки, утилиты и среды программирования, средства пакетного выполнения процедур"]
```

**Тест Б2 (положительный)**
- Описание: Проверка корректности работы метода, находящего для введенной формулировки схожие формулировки из списка формулировок.
- Метод: *find_similar_formulations_v2(formulation, formulations)*.
- Входные данные: *formulation*, *formulations*
```
formulation = "Методы коммуникаций"
  
formulations = ["Методы коммуникации и что-то там еще", "Средства коммуникаций и что-то там еще"]
```
- Ожидаемый результат: ["Методы коммуникации и что-то там еще"]
